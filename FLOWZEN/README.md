Flowzen - AI Productivity & Habit Tracking Platform

An AI-powered productivity SaaS platform designed to help users manage tasks, track habits, and improve consistency through smart analytics and visual insights.

Features
Smart To-Do Lists: Organize daily tasks efficiently with a clean and modern interface.
AI Productivity Tracking: Analyze task completion patterns and productivity trends using AI.
Habit Tracker: Monitor daily habits and maintain consistency over time.
Performance Analytics: Visual graphs and pie charts to track progress and productivity.
Progress Insights: Get detailed insights into completed and pending goals.
Subscription Plans: Multiple pricing tiers with advanced productivity features.
Cloud-Based Access: Access your workspace anytime through the web platform.
Deployment

To deploy this application, follow these steps:

1. Environment Variables

Configure the following environment variables in your deployment platform:

Variable	Description
VITE_API_URL	Backend API URL
DATABASE_URL	Database Connection URL
AI_API_KEY	AI Service API Key
AUTH_SECRET	Authentication Secret Key
2. Build Settings

Ensure the following build settings are configured correctly:

Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
3. Install Dependencies
npm install
4. Run Development Server
npm run dev
Tech Stack
Frontend: React + TypeScript + Vite
Styling: Tailwind CSS
Backend: Node.js / Express
Database: MongoDB / PostgreSQL
AI Integration: Gemini API / OpenAI API
Authentication: Firebase / Supabase / JWT
Charts & Analytics: Chart.js / Recharts
